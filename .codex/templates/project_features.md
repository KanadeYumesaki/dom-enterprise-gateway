# このシステムでできること（かんたん説明）

- 生成日時: 2025-12-06
- 生成モデル: GPT-5 (Codex, Antigravity 拡張)

## 主な機能
- チャット: AI と会話し、ファイル添付も可能。
- ナレッジ閲覧: 管理者がアップロードした社内ドキュメントを閲覧。
- 設定: テーマ / 言語 / 文字サイズ / オンボーディング状態を変更。
- ヘルプ: 使い方のヒントを一覧と詳細で確認。

## 使い方の流れ
1. ログインする。
2. チャットやナレッジを試す。
3. 必要に応じて `/settings` で表示を自分好みに変える。
4. 困ったら `/help` を開き、記事を読む。

## 安全利用の注意
- 社外秘のデータは入力前にルールを確認してください。
- ブラウザを閉じる前にログアウトすることを推奨します。
